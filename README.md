## Why Astra?

I built Astra because I was tired of bots that lock basic features behind paywalls. You shouldn't need to pay $10/month just to use music or level roles.

**Everything is free. Forever.** No "premium", no "pro tier", no subscriptions.

## Tech Stack

For the curious:

- **Bot**: TypeScript, Discord.js v14
- **API**: Express, Passport (Discord OAuth2)
- **Dashboard**: React 18, TailwindCSS, Zustand, React Query
- **Database**: MongoDB with Mongoose
- **Hosting**: Pelican.dev (single-port architecture)

Built to be fast, reliable, and maintainable.
